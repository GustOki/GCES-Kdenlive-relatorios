# Relatório de Contribuição – Sprint 0

**Disciplina:** Gestão de Configuração e Evolução de Software
**Equipe:** Equipe Kdenlive
**Comunidade/Projeto de Software Livre:** Kdenlive
**Período da Sprint:** 06/04/2026 – 21/04/2026

---

## 1. Resumo da Sprint

Sprint inicial de planejamento e configuração do repositório de documentação da disciplina GCES. Nesta sprint, a equipe se organizou para trabalhar com o projeto **Kdenlive**, um editor de vídeo não-linear de código aberto. As principais atividades foram a criação e configuração do repositório de documentação, o entendimento do fluxo de contribuição do projeto e o estudo do ambiente de desenvolvimento.

---

## 2. Objetivos da Sprint

- [x] Criar e configurar o repositório de documentação com MkDocs.
- [x] Definir os membros da equipe.
- [x] Escolher e documentar o projeto de software livre: Kdenlive.
- [x] Criar materiais de apoio (templates, guias de MR).
- [x] Configurar CI/CD para publicação automática via GitHub Pages.
- [x] Estudar o ambiente de desenvolvimento do Kdenlive.
- [x] Realizar a leitura inicial da documentação de contribuição do Kdenlive.
